Project

Temperature Simulation System with LED Indicators

Project objective

Design an electronic system capable of detecting temperature changes and displaying the level through visual indicators and an audible alert.

Technical description

A small-scale dome was built that integrates a temperature sensor connected to an LED indicator system. The circuit activates different LEDs according to the detected thermal level and emits an audible alert when the defined maximum temperature is reached. The system was implemented through electronic integration and calibration of activation thresholds.

Technologies used
  • Temperature sensor
  • Analog electronics
  • LED indicators
  • Audible alert system
  • Electronic circuit integration
My specific role

System design, integration of electronic components, and functional testing.

Results obtained

The temperature level was visualized via luminous indicators and an audible alert when the maximum value was reached.
