Project

LED sequence with 555 timer

Project objective

Design an analog electronic circuit capable of generating an automatic LED lighting sequence without the use of programming.

Technical description

A sequential circuit was built using 555 timers configured as periodic signal generators. The system was fully wired with analog components, achieving a lighting sequence similar to decorative lights. Operation was obtained through electronic timing control without microcontrollers.

Technologies used
  • 555 timer integrated circuits
  • Analog electronics
  • Circuit design and wiring
  • Electronic timing control
My specific role

Circuit design, component selection, circuit construction, and functional testing.

Results obtained

A stable automatic lighting sequence was obtained using only analog electronics.
